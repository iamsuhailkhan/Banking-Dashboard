# Banking Dashboard (MERN Stack)

A full‑stack banking dashboard built with **MongoDB, Express, React, Node.js (MERN)**.  
Features include JWT authentication, role‑based access (Admin vs User), protected APIs, and an Admin Dashboard with charts.

---

## 🚀 Features
- User signup/login with JWT authentication
- Role‑based access control (Admin vs User)
- Protected transaction APIs
- User Dashboard → personal transactions + expense chart
- Admin Dashboard → all transactions + stats + bar/pie charts
- Responsive UI with SASS + GSAP animations
